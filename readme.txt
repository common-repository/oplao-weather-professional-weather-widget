== Description ==
Oplao.com - Professional weather forecast. We've created this widget to provide easy to use, fast, accurate weather forecasting tool for your website.

Wat's inside? 
* Fast live search
* Current weather
* Up to 5 day weather forecast. You can use 3, 5 day weather forecast of enjoy current one.
* 5 beautiful design theme
* 4 responsive dimensions, suitable for any WordPress website. 

If you Like our Widget - please rate it with 5 star. Thank You!

Use the built in widget with all of its marvelous settings or add it to a page or theme with the shortcode: (all settings shown)

[oplao-weather location="Minsk" units="C" size="med or min or wide" forecast_days="3" hide_stats="1" inline_style='max-width: 300px']`

= Settings =

*   Location: Enter a string like "Minsk, Belarus" or just "London".
*   Units: C or F (default)
*   Size: Min (default) or wide or med
*   Forecast Days: How many days to show in the forecast bar (3 or 5)
*   Inline style: styles for widget

== Installation ==

1. Add plugin to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use  widget to display oplao weather widget on your site
